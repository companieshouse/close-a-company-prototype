Application signed

The person making the application should notify you once it has been submitted.

We need to receive signatures from all signing directors before the person making the application can pay for and submit it. It may take some time for others to sign. You can check who has signed below.

What to do next

Once the application has been submitted, you will be able to download a signed PDF copy by signing back in to the Apply to strike off and dissolve a company service. https://find-and-update.company-information.service.gov.uk/close-a-company/search-company.

The directors must send this to all interested parties within 7 days of the application being submitted.

! Warning It is a criminal offence not to tell all interested parties that an application has been made to close a company.

Read the full list of interested parties you must tell about the company closing (opens in a new tab).
